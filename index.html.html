📸 Google Drive Photo Gallery Setup

Photos from your Drive folder will auto-show here. If you delete them from Drive, they'll be removed from here too.

Setup Steps (only needs to be done once):
  1. Create a Google Drive folder — go to drive.google.com, create a new folder (e.g. "Website Photos"), and add your photos to it.
  2. Make the folder public — Right-click the folder → Share → General access: Anyone with the link → Viewer → Done.
  3. Copy the Folder ID — Open the folder and look at the URL:
    drive.google.com/drive/folders/ID_IS_HERE
    Copy that long ID.
  4. Create a Google API Key — go to console.cloud.google.com/apis/credentials:
    • Create a new project (if you don't have one)
    • Left menu → Enabled APIs → + ENABLE APIS AND SERVICES → search Google Drive API → Enable
    • Go back to Credentials → + CREATE CREDENTIALS → API key → copy it
    • (Optional but recommended) Edit the API key → Application restrictions: HTTP referrers → add your domain
  5. Paste it below and Save ✅
⚡ Auto Sync: Photos are fetched from Drive every time the page opens. If you delete them from Drive, they won't show here either. Add a new photo to Drive → press the Refresh button or reload the page.

🔗 Google Sheet Project Sync Setup

Jab aap yahan project Add/Edit/Delete karenge, wo is Sheet mein bhi save hoga — aur jo bhi aapka link kholega usko wahi latest projects dikhenge (localStorage nahi, sabke liye same data).

Setup Steps (sirf ek baar karna hai):
  1. Google Sheet banayein — sheets.google.com par jaake ek naya Sheet banaye.
  2. Apps Script kholein — Sheet ke upar menu se Extensions → Apps Script par click karein.
  3. Code paste karein — Jo code Claude ne niche chat mein diya hai, use copy karke Apps Script editor mein (default Code.gs ke andar sab select karke delete karke) paste kar dein, phir 💾 Save karein.
  4. Deploy karein — Upar right side Deploy → New deployment → gear icon ⚙️ se type select karein Web app:
    • Execute as: Me
    • Who has access: Anyone
    • Deploy par click karein, Google account permission allow karein.
  5. Web app URL copy karein — deploy hone ke baad ek URL milega jaisa https://script.google.com/macros/s/XXXXXXXX/exec, wahi copy karein.
  6. Neeche paste karein aur Save karein ✅
⚡ Auto Sync: Page khulte hi latest projects Sheet se load honge. Jab aap koi project add/edit/delete karenge, wo turant Sheet mein bhi update ho jayega — bas internet chahiye.
